COMMON PATTERNS ACROSS ALL SAMPLES
====================================================================

1. URGENCY TACTICS (Present in 5/5 samples - 100%)
   - Artificial deadlines (24-72 hours)
   - Account suspension threats
   - Financial penalties
   - Time-sensitive opportunities

2. FAKE/SPOOFED DOMAINS (5/5 samples - 100%)
   - Typosquatting (fedx vs fedex)
   - Similar domains (company-corp.com vs company.com)
   - Suspicious TLDs (.tech, .delivery)
   - Recently registered domains

3. AUTHORITY EXPLOITATION (4/5 samples - 80%)
   - CEO impersonation
   - IT department impersonation
   - Security team impersonation
   - Vendor/supplier impersonation

4. SOCIAL ENGINEERING (5/5 samples - 100%)
   - Fear tactics
   - Creating artificial problems
   - Offering solutions that compromise security
   - Exploiting trust in authority

5. CREDENTIAL HARVESTING (4/5 samples - 80%)
   - Fake login pages
   - Password reset scams
   - Account verification requests
   - Financial information requests

6. GENERIC COMMUNICATION (4/5 samples - 80%)
   - "Dear Customer" / "Dear Employee"
   - No personalization
   - Mass distribution indicators
   - Lack of specific account details

KEY LEARNING OBJECTIVES
====================================================================

After studying these samples, users should be able to:

✓ Identify suspicious sender domains
✓ Recognize urgency and fear-based tactics
✓ Spot generic vs. personalized communications
✓ Verify links before clicking (hover technique)
✓ Identify double file extensions (.xlsx.exe)
✓ Recognize authority exploitation attempts
✓ Understand why verification is crucial
✓ Know how to report phishing attempts
✓ Implement security best practices
✓ Respond appropriately to suspected phishing
